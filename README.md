# Rick and Morty Frontend Challenge

Aplicación web desarrollada como ejercicio técnico frontend. Permite consultar personajes de Rick and Morty, ver su detalle en un modal y gestionar una lista de favoritos.

## Demo

Pendiente de agregar URL de deploy.

## Stack técnico

- React
- TypeScript
- Vite
- Context API
- CSS Modules / CSS por componente
- Vitest
- Testing Library

## Funcionalidades implementadas

- Consumo de Rick and Morty API.
- Renderizado inicial de personajes.
- Modal de detalle por personaje.
- Agregar personajes a favoritos.
- Eliminar personajes de favoritos.
- Sección independiente de favoritos.
- Buscador por nombre.
- Paginación.
- Persistencia de favoritos en localStorage.
- Tests unitarios para componentes y cliente de API.
- Diseño responsive.
